Tareas de mantenimiento y desarrollo a sistemas de recursos humanos

Realizar tareas de recolección de requisitos de software, análisis y diseño, bases de datos, implementación y paso a producción de un sistema de gestión de RTO..

Funciones desarrolladas:

  • ABM de RTO.
  • ABM de roles y permisos.
  • Generación de certicados PDF con qr y marca de agua para verificar autenticidad.
  • Instalación de servidor Apache.
  • Puesta en producción en servidor linux.

Elaboración de documentación:

  • Manual de uso.
  • Diseño de digrama de CU, DER necesarios.
  • Uso de tablero kamban para llevar seguimiento de tareas.

Tecnologías usadas:

  • Framework Laravel
  • PHP
  • PostgreSQL
  • Javascript
  • HTML
  • CSS
