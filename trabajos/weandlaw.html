Desarrollo de aplicaciones móviles híbridas

Desarrollo de aplicaciones móviles híbridas a partir de sistemas web.

Funciones desarrolladas:

  • Implementación de web en aplicación hibrida, para lo cual utilicé el componente nativo de Android webview que permite usar interfaz javascript para poder interactuar con la web embebida.
  • Creación de iconos e imagenes adaptables.
  • Creación de imagen de inicio de aplicación.

Otras tareas:

  • Subir la app a la playstore.
  • Mantenimiento y actualizaciones.
  • Elaboración de video de funcionalidades.

Tecnologías usadas:

  • Consola de desarrollador de Google Play
  • Android Studio
  • Kotlin
  • Javascript
  • Wordpress
