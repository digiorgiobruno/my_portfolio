Creación de e-learning con PHP

Trabajo freelance en el cual desarrollé una plataforma dinámica de venta de cursos online integrando Mercadopago y Vimeo.

Funciones desarrolladas:

  • Diseño frontend, backend y bases de datos. Migrar al servidor en la nube usando FTP.
  • ABM de cursos, categorias, credenciales de pago, datos de usuario etc.
  • Login y registro con recuperación de contraseña.
  • Sistema de puntaje por estrellas.
  • Control de visualizaciones.
  • Agregado de campos nuevos, archivos adjuntos, etc..
  • Integración de Mercadopago y Paypal.

Módulos desarrollados:

  • Módulo de ABM de cursos.
  • Módulo de Configuración de credenciales de Mercadopago y Paypal.

Tecnologías usadas:

  • PHP
  • MySQL
  • Javascript
  • HTML
  • CSS
