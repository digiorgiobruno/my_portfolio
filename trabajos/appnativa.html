Desarrollo de app nativas Android

Como trabajo final de la materia Ingeniería de Software desarrolle una app de creación de eventos sociales usando Kotlin, Android Studio y Firebase.

Funciones desarrolladas:

  • ABM de eventos con base de datos en tiempo real de firebase.
  • Implementación de un feed con el uso de recyclerview vertical.

Otras tareas:

  • Desglose de tareas, diseño e implementación del software, gestión de tareas usando SCRUM, estimación usando scrum poker.
  • Elaboración de documentación como diagramas de clases, DER, Historias de usuario, etc.

Tecnologías usadas:

  • Kotlin
  • Android Studio
  • Firebase
  • Jira
