Tareas de mantenimiento y desarrollo a sistemas de recursos humanos

Las tareas realizadas van desde la corrección de errores hasta el desarrollo de nuevas funcionalidades o módulos completos.

Funciones desarrolladas:

  • Extracción de datos de PDF de recibos de sueldo.
  • ABM conceptos de recibos.
  • Funcionalidad para agregar texto configurable de vacaciones al pdf generado.
  • Agregado de campos nuevos, archivos adjuntos, etc..
  • Mejoras estéticas.

Módulos desarrollados:

  • Sistema de mensajería entre el sistema principal y el sistema de mesa de ayuda.
  • Módulo de novedades en página principal.
  • Desarrollo y puesta en producción de aplicación móvil híbrida a partir del sistema de gestión de recursos humanos .

Elaboración de documentación:

  • Manual para subir apps a la google play y activar notificaciones push.
  • Manual de texto configurable de vacaciones.
  • Manual para hacer que los cuadros tengas columnas y filas fijas.

Tecnologías usadas:

  • Framework SIU TOBA
  • PHP
  • PostgreSQL
  • Javascript
  • HTML
  • CSS

Tecnologías usadas para desarrollo móvi:

  • Android Studio
  • Kotlin
  • Consola de desarrollador de Google Play
  • Consola de desarrollador de Firebase
